# Client (Frontend)

This directory contains the frontend code for **RideShare UMass**, a student-only carpool and ride-sharing platform.

The client is responsible for:
- User authentication UI
- Ride creation and search flows
- Dashboards for drivers and riders
- Messaging and notifications
- Mobile-responsive user experience

## Planned Tech Stack
- React.js or Next.js
- Tailwind CSS
- REST API integration with backend
- Map integration (Google Maps / OpenStreetMap)
